

# Secure Note App

#### A secure note-sharing application where users can create password-protected notes and share them safely using a unique URL.

The password is generated by the backend, securely hashed before storing in the database, and shown only once to the user.

## Setup Instructions
### Clone the repo
```
https://github.com/Rohitsinghkhetwal/Private-Note-Sharing-.git

```
### Backend Setup
```
cd backend
npm install 

```
Create a .env file inside backend folder:
```
PORT  = 8000

MONGODB_URL=mongodb//<YOUR_MONGODB_URL>

GEMINI_API_KEY=<YOUR_GEMINI_API>

GEMINI_MODEL=gemini-2.5-flash

FRONTEND_URL=localhost:3000
```
### Start Backend Server
```
npm run dev
```

## Frontend Setup
```
cd frontend
npm install
npm run dev
```

```
http://localhost:3000
```
